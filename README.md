# Nixploit 

## Usage

On the host, make sure the firewall allows DHCPv4 requests from the Incus bridge
`nixploit-net` to the host DHCP server, i.e. UDP destination port 67. 

```bash
chmod +x maker.sh
./maker.sh
```
